# notebookLM_financeiro_DIO
Um NotebookLM, que reúne cinco fontes abertas em texto ou PDF sobre Reserva Financeira de Emergência. A partir desse material, são definidos objetivos de estudo eelaboradas perguntas estratégicas, assim possuindo um miniguia de estudo com resumos estruturados, glossário de conceitos e um conjunto de prompts reutilizáveis.
